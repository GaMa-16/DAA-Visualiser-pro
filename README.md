## 🖋️ The Vision
[cite_start]Built specifically for the **SRMIST 21CSC204J: Design and Analysis of Algorithms** curriculum, this visualizer is designed to move students from passive reading to active analysis[cite: 3]. By combining a "Hand-Drawn" aesthetic with high-performance visualizations, it lowers the barrier to entry for complex algorithmic paradigms.

## 🚀 Key Features

### 1. Interactive Algorithm Lab 🧪
Visualize every paradigm from the syllabus with "Step-Back" debugging and wobbly UI elements:
* [cite_start]**Divide & Conquer:** Binary Search, Quick Sort, Merge Sort, and Strassen’s Matrix Multiplication[cite: 10].
* [cite_start]**Optimization (Greedy & DP):** Huffman Coding, Knapsack variations, and Longest Common Subsequence (LCS)[cite: 13].
* [cite_start]**Constraint Satisfaction:** N-Queens, Sum of Subsets, and Travelling Salesman Problem (TSP) using Backtracking and Branch & Bound[cite: 15].

### 2. AI-Powered DAA Tutor 🤖
Integrated with **Gemini Intelligence**, the built-in chatbot acts as a context-aware teaching assistant:
* [cite_start]**Recurrence Coaching:** Get step-by-step breakdowns using the **Substitution method** or **Recursion tree**[cite: 7].
* [cite_start]**Mathematical Rigor:** Automated complexity analysis using **Master Theorem** examples[cite: 10].
* [cite_start]**Syllabus Sync:** Explanations tailored to SRMIST's Learning Outcomes (CO-1 to CO-5)[cite: 3].

### 3. Performance Benchmarking 📈
[cite_start]Compare the "Growth Functions" of different algorithms in real-time[cite: 6].
* [cite_start]Plot growth curves for **Best, Worst, and Average case** scenarios[cite: 6, 10].
* [cite_start]Visualize the difference between polynomial time and exponential time problems[cite: 3].

## 🎨 Design System
This project strictly follows the **Hand-Drawn Design Philosophy**:
* **The "No Straight Lines" Rule**: Every container uses irregular `borderRadius` to create organic, wobbly edges.
* **Authentic Texture**: A paper-grain dot pattern background simulates a physical notebook.
* **Hard Offset Shadows**: No blur—just solid, pencil-black offsets for a cut-paper aesthetic.
* **Human Typography**: Uses **Kalam** for marker-style headings and **Patrick Hand** for body text.
